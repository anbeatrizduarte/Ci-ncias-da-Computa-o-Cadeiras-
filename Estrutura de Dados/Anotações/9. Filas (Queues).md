
#### Sobre

Uma fila é uma estrutura de dados do tipo **FIFO** (First In, First Out), ou seja, o primeiro elemento a entrar é o primeiro a sair. Diante disso, temos o exemplo de filas de bancos, ou qualquer estabelecimento.

- Inserção:

		r := prov
